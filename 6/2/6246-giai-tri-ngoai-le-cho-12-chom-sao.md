## Giải trí ngoài lề cho 12 chòm sao ^^

## conan97k

Mọi người thử xem đúng k và nghĩ cách giải thích nhé​
- Đếm số chữ cái trong họ tên cuả bạn ( đầy đủ nhé )
- Lấy số đó nhân ( x ) với 2
- Tiếp tục cộng ( + ) với 5
- Rồi nhân ( x ) với 50
- Sau đó cộng ( + ) với 1762
- Rồi trừ ( - ) đi năm sinh cuả bạn
=> 2 số cuối cuả kết quả chính là tuổi cuả bạn​
