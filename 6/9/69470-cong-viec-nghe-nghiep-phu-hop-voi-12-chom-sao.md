## Công việc Nghề nghiệp phù hợp với 12 chòm sao.

## Alice CanCer

Bạch Dương : Bộ đội ( Dũng cảm, sẵn sàng hy sinh )
Kim Ngưu : Nhà chính trị ( Nhiều ý tưởng kiếm tiền )
Song Tử : MC ( Tài lanh, nhanh mồm nhanh miệng )
Cự Giải : Diễn viên ( Mau ra nước mắt )
Sư Tử : Võ Sĩ ( Sinh ra vốn khỏe mạnh và có lòng quyết tâm )
Xử Nữ : Bất động sản ( Tính tình cẩn thận )
Thiên Bình : Người mẫu ( Có mắt thẩm mĩ cao )
Thiên Yết : Điệp viên ^ ^( Bí ẩn, trực giác nhạy bén )
Ma Kết : Giám đốc ( Có khả năng chỉ đạo )
Nhân Mã : Giáo viên mầm non ( Tính tình trẻ con )
Bảo Bình : Nhà bác học ( Nhiều ý tưởng )
Song Ngư : Thi sĩ ( Hay mơ mộng nên có nhiều ý tưởng )
