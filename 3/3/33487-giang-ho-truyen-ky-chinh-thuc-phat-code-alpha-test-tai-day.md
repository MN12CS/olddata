## Giang Hồ Truyền Kỳ chính thức phát code alpha test tại đây

## jibunnie

Sau rất nhiều ngày chờ đợi cuối cùng thời Alpha test của Giang Hồ Truyền Kỳ đã được bật mí. Các game thủ sẽ không còn phải “tấn công Fan page điên cuồng” để xin code test nữa. Ban Quản Trị xin tuyên bố chính thức phát code test Giang Hồ Truyền Kỳ.
 
 

Đúng 10h sáng ngày 14/03/2013 tới, Giang Hồ Truyền Kỳ sẽ trình làng Game thủ nước nhà với phiên bản Alpha test, các Game thủ sẽ có 6 ngày để thỏa sức tìm hiểu các tính năng hấp dẫn của Game trước khi bước vào phiên bản Open beta
 

 
 ​ 
 
 
Ban Quản Trị sẽ phát tận tay Code test đến tay các bạn thông qua Fanpage chính của Game , Forum chính của game email: contact@gianghotruyenky.com, Yahoo Hỗ Trợ 1,2,3 trên trang chủ  , các số Mobile Hỗ Trợ . Đồng thời cũng sẽ phát code trên các diễn đàn Game, các trang tin điện tử về Game, và qua các sự kiện trên trang chủ. Ngay từ bây giờ các bạn hãy nhanh chóng để mail hoặc Yahoo để nhanh  nhanh tay nhận code test Giang Hồ Truyền Kỳ !!! 
 
 
 
Với bản Alpha test lần này, Game thủ sẽ được trải nghiệm với một số trong vô vàn tính năng đặc sắc của Game như : kháng kim anh hùng, bình định phản loạn, các chiến dịch phụ bản… Đặc biệt phủ bản : Gangnam Style sẽ là điều thú vị bất ngờ nhất dành cho tất cả các Game thủ.
