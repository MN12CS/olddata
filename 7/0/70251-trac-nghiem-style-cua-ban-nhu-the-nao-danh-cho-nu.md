## Trắc nghiệm Style của bạn như thế nào ? ( Dành cho nữ )

## Alice CanCer

Style của bạn ntn ???  
1. Trang phục :
a ) Bạn thích những bộ trang phục điệu đà, nhẹ nhàng và nữ tính
b ) Bạn thích những bộ trang phục truyền thống
c ) Bạn thích những bộ trang phục xì-teen, bụi bặm

2. Trang sức :
a ) Bạn thích những đồ trang sức óng ánh và sặc sỡ
b ) Bạn thích những đồ trang sức tự làm
c ) Bạn thích những đồ trang sức đắt tiền 

3. Trang điểm :
a ) Khi trang điểm, bạn thích những màu sắc nhẹ nhàng, nữ tính ( VD : hồng, vàng,... )
b ) Khi trang điểm, bạn thích những màu trẻ trung, tươi mới ( VD : cam, xanh,... )
c ) Khi trang điểm, bạn thích những màu sắc đậm đà, cuốn hút ( VD : tím, đỏ,... )

4. Nếu như mai sau bạn kết hôn, bạn sẽ chọn váy cưới kiểu gì ?
-  Màu sắc :
a ) Màu trắng tinh khiết
b ) Màu hồng điệu đà
c ) Màu đỏ cuốn hút
- Kiểu dáng :
a ) Váy ngắn, gọn gàng
b ) Váy cưới truyền thống
c ) Váy hở lưng, dài qua đầu gối

* Đáp án :
- Nếu bạn hầu hết chọn a : Bạn là một người khá hiền lành nhưng không kém phần điệu đà, tính bạn rất thoải mái khiến cho người đối diện cảm thấy dễ chịu. Bạn rất dễ bị tổn thương và là một người dễ gần. Nhưng đôi khi, bạn hay bị mặc cảm và tự ti vì những sự cố không theo như ý muốn.
- Nếu bạn hầu hết chọn b : Bạn là một người tiết kiệm và có một tâm hồn đẹp. Bạn không thích những món đồ mua ngoài chợ, bạn thích những món đồ tự làm và thích những bộ quần áo truyền thống. Nhưng đừng tiết kiệm quá nhé, mọi người sẽ nghĩ bạn là người sống cứng nhắc và gò bó đấy.
- Nếu bạn hầu hết chọn c : Bạn là một con người tự do, tự tại. Bạn không thích làm một cô gái nữ tính, dịu dàng, bạn thích những gì mới mẻ và sống động. Bạn khá phóng khoáng trong chuyện tiền bạc cũng như chuyện tình cảm, bạn rất hòa đồng nên mọi người rất thích chơi với bạn đấy !
P/s : Mọi người có ghé thăm thì nàm ơn quẳng cho em một cái like *van xin*
