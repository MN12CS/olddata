Kim Ngưu: khi nó nói // Mang đồ ăn đến đây đi // là thực ra nó đang nghĩ // Tui đang cần được yêu thương gần chết đây nè

Song Tử: khi nó im lặng // là thực ra nó đang nghĩ // Cậu vừa tổn thương tui đó

Cự Giải: khi nó nói // Thôi quên đi // là thực ra nó đang nghĩ // Liệu hồn mà sửa lỗi đi

Sư Tử: khi nó nói // Cậu chẳng là gì với tui hết // là thực ra nó đang nghĩ // Với tui cậu có ý nghĩa lớn lắm đó trời ơi

Xử Nữ: khi nó nói // Chuyện đó thiệt nhảm nhí // là thực ra nó đang nghĩ // Chuyện đó quan trọng bome đi được trời ơi

Thiên Bình: khi nó nói // Tui mệt quá đi // là thực ra nó đang nghĩ // Ôm tui một cái được hông

Thiên Yết: khi nó nói // Chuyện đó chả quan trọng nữa // là thực ra nó đang nghĩ // Chuyện đó quan trọng vậy cơ mà, tui đang tổn thương lắm đó nhưng đừng mơ tui sẽ thừa nhận với cậu

Nhân Mã: khi nó nói // Không sao mà haha // là thực ra nó đang nghĩ // Lo mà xin lỗi đi chứ

Ma Kết: khi nó nói // Giết tui luôn đi cho rồi // là thực ra nó đang nghĩ // Làm ơn hỏi tui có sao hông đi mà

Bảo Bình: khi nó nói // Để tui yên giùm đi // là thực ra nó đang nghĩ // Làm ơn chú ý đến tôi đi
